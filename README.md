# Mobile App Starter (Angular + Ionic + Capacitor + SQLite + Bootstrap)

This project is a mobile app starter template using the following stack:

- Angular
- Ionic
- Capacitor
- SQLite (via @capacitor-community/sqlite)
- Bootstrap (with SCSS)
